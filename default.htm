XMPL Sample Landing Page

Getting started using Dreamweaver and uCreate Digital

    1. Open Adobe Dreamweaver with XMPie uCreate Digital Plugin.
    2. Using the XMPie uCreate Digital Panel, select “Link to Project”.
    3. Once the Panel is connected, you can edit the site content and tag it with XMPie personalization.


Getting started using other web editors

    1. Login to Circle and select the Project.
    2. In the Project Library, go to “Website” and download the configuration file.
    3. Place the Configuration File in your local copy of the web site.
    4. Edit the site content and tag it with XMPie personalization.
    5. Click here to learn how to add the XMPL language.
